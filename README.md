# 📚 PDF Learning Platform

A simple learning platform that displays PDFs based on selected subjects.

## 🚀 Features
- View subject-based PDFs
- Simple UI
- Fast and lightweight

## 📁 Project Structure

- `app/page.js` → Main frontend
- `public/pdfs/` → Store all PDFs here
